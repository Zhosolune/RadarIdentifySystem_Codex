# PyQt6 重构阶段索引（一个阶段一个文件）

## 阶段顺序
1. P00 - 重构约束冻结与台账初始化（含 Fluent 许可确认）
2. P01 - 新工程骨架、FluentWindow 与 `app/config.py` 初始化（含 plot 基础配置项）
3. P02 - 全局信号总线（`app/signal_bus.py`）
4. P03 - Core 数据预处理与切片
5. P04 - Core 聚类流程
6. P05 - Core 识别与参数提取
7. P06 - Core 合并规则与结果模型
8. P07 - Infra 适配层（解析/推理/绘图/导出）
9. P08 - App 工作流与状态
10. P09 - UI 主链路迁移（Fluent 导航 + plot 配置应用）
11. P10 - UI 高级功能迁移（SettingCard 与配置页，含 plot 全量参数编辑）
12. P11 - 全速处理与线程治理
13. P12 - 测试打包与发布切换（含 Fluent 合规复核）

## 强制同步文档
- `docs/目录结构与分层约束.md`
- `docs/重构接口对接手册.md`
- `docs/重构执行追踪.md`
- `docs/配置系统设计.md`（配置字段变更时）
- `docs/PyQtFluentWidgets可行性评估.md`（UI策略变更时）

## 执行规则
1. 每阶段完成后再进入下一阶段。
2. 每阶段记录：完成项、未完成项、回归结果、回退点。
3. 允许优化，但必须证明“结果等价”。
4. 当前重构不引入 `db` 层；若出现落库需求需先更新架构文档再实施。
